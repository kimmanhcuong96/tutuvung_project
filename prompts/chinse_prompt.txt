- Generate exactly 2 Chinese vocabulary words randomly at CEFR HSK2-HSK4.

3) meaning (Vietnamese, exactly 1 meaning)

   - word: related word/phrase derived from main character (multi-character allowed)

Requirements for other_meaning:
- Include 5–9 meanings (prefer 8–9)
- Maximize part-of-speech diversity (noun, verb, adj, adv, etc.)
- Include pronunciation differences if the character is 多音字
- Avoid repeating similar meanings
- Use distinct compound words for different meanings if single-character usage is limited

Output format:
- **Return JSON as a pure array** (no object wrapper, no "data")
- **Output in 2 separate messages**:
   1) First message: JSON array of the 2 words with other_meaning
   2) Second message: JSON array of excluded_words, starting with ["你","我"] plus the 2 new words

- **Return JSON only**. No explanations. No extra text.
